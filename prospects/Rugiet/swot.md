- **Stack alignment:** Posting centers **Rails + React + React Native + PostgreSQL**—a trifecta many candidates lack; strong match for full-stack/mobile generalist profile.
- **AI-native hiring signal:** Explicit requirement for **AI-assisted development** favors engineers already shipping with **Cursor**/LLM tooling daily.

## Threats

- **Competition:** Ro, Hims, and other telehealth incumbents with larger eng budgets and brand trust.

- **Candidate bar on mobile:** "Own end-to-end mobile" may filter for engineers with recent **store release** ownership over strong RN feature delivery.
